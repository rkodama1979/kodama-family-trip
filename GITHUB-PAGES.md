# GitHub Pages 公開手順

1. GitHubへログインします。
2. 右上の「＋」から「New repository」を選びます。
3. Repository nameに `kodama-family-trip` と入力します。
4. Publicを選び、「Create repository」を押します。
5. 「uploading an existing file」を押します。
6. このZIPを解凍したフォルダの中身をすべてドラッグ＆ドロップします。
   - `index.html`
   - `compare.html`
   - `vote.html`
   - `css`
   - `js`
   - `images`
   - `pages`
7. 「Commit changes」を押します。
8. リポジトリ上部の「Settings」→左側の「Pages」を開きます。
9. Sourceを「Deploy from a branch」にします。
10. Branchを `main`、フォルダを `/ (root)` にして保存します。

公開URL：
`https://rkodama1979.github.io/kodama-family-trip/`

反映には数分かかる場合があります。
